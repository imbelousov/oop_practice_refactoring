# Валидатор моделей

В этом задании не будет конкретных требований к результату.
Класс `Validator` выполняет проверку значений свойств произвольной модели данных
на обозначенные с помощью атрибутов требования. Однако разработчик торопился, и у него не было
времени подумать над организацией кода. Произведите рефакторинг класса `Validator`, применив
полученные в рамках курса знания о лучших практиках проектирования в парадигме ООП.
